# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-01-01

### Added
- Initial release of mcp-server-aws.
- Tool: `ec2_list_instances` - List EC2 instances with optional filters.
- Tool: `s3_list_buckets` - List all S3 buckets in the account.
- Tool: `s3_get_object` - Retrieve an object from an S3 bucket.
- Tool: `lambda_invoke` - Invoke an AWS Lambda function.
- Tool: `cloudwatch_get_metrics` - Retrieve CloudWatch metrics data.
- Tool: `iam_list_roles` - List IAM roles in the account.
- Tool: `sts_get_caller_identity` - Get the identity of the current caller.
- MCP server with stdio transport.
- Docker support.
